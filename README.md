## ğŸš€ Getting Started

### Prerequisites

Make sure you have installed:

* Flutter SDK
* Android Studio / VS Code
* Android Emulator or Physical Device

Check Flutter installation:

```
flutter doctor
```

### Installation

1. Clone the repository

```
git clone https://github.com/your-username/breakfast-app-flutter.git
```

2. Go into the project directory

```
cd breakfast-app-flutter
```

3. Install dependencies

```
flutter pub get
```

4. Run the app

```
flutter run
```
